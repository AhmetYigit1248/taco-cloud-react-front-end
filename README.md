# Taco Cloud - Frontend

React frontend for the Taco Cloud app. Connects to the Spring Boot backend API.

## Tech Stack

- React 19
- React Router 7

## Pages

- Home, Login, Registration
- Taco designer - pick ingredients, build your taco
- Order form and order history
- Admin panel for user/order management
- Protected routes for authenticated users

## Run

```
npm install
npm start
```
